📍 ENDEREÇAMENTO IP

Juninho, o que o endereço IP representa na rede para um dispositivo?

📚 RESUMO PARA O JUNINHO:

O **IP (Internet Protocol)** é como o CEP da sua casa. Sem ele, os pacotes de dados não saberiam onde entregar a informação. Cada dispositivo na rede precisa de um IP único para ser encontrado!

📦 PROTOCOLO TCP

Qual é a principal característica do protocolo TCP no envio de dados?

📚 RESUMO PARA O JUNINHO:

O **TCP** é o protocolo "confiável". Ele estabelece uma conexão e verifica se cada pedaço do arquivo chegou certinho. Se algo faltar, ele pede para reenviar. É o que usamos para baixar arquivos e ver sites!

📖 O DNS (SISTEMA DE NOMES)

Juninho, qual é a função do servidor DNS na internet?

📚 RESUMO PARA O JUNINHO:

O **DNS** é a "Lista Telefônica" da internet. Seria impossível decorar o IP de cada site. O DNS faz o trabalho duro: você digita `google.com` e ele te diz que o IP é `8.8.8.8`!

🤖 O MÁGICO DHCP

O que acontece quando o Juninho conecta um celular no Wi-Fi e ele recebe um IP sozinho?

📚 RESUMO PARA O JUNINHO:

O **DHCP** é quem automatiza a rede! Em vez de você configurar o IP na mão em cada aparelho, o servidor DHCP (geralmente o roteador) "empresta" um IP livre para quem acaba de chegar. Praticidade total!
