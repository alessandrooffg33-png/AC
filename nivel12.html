O CORAÇÃO DA REDE

Juninho, para que serve o Active Directory (AD) em um servidor Windows?

📚 RESUMO PARA O JUNINHO:

O **Active Directory** é o banco de dados da empresa. Em vez de você criar um usuário em cada PC, você cria no AD uma vez só, e o funcionário pode logar em qualquer computador da rede com a mesma senha!

O DOMÍNIO

Como se chama o "território" que o servidor controla (ex: empresa.local)?

📚 RESUMO PARA O JUNINHO:

O **Domínio** é o nome da rede controlada pelo servidor. Quando um PC "entra no domínio", ele passa a obedecer às ordens do servidor Windows Server. É o nível profissional de organização!

O PODER DAS GPOs

O que são as GPOs (Group Policy Objects) que o Juninho configura?

📚 RESUMO PARA O JUNINHO:

As **GPOs** são as leis da rede. Quer bloquear o Painel de Controle? Quer mudar o papel de parede de todos os 100 PCs ao mesmo tempo? Você usa uma GPO! É o controle total na mão do Analista.

ORGANIZANDO TUDO

Como o Juninho separa os usuários de "Financeiro" dos de "Vendas" no servidor?

📚 RESUMO PARA O JUNINHO:

As **OUs (Organizational Units)** funcionam como pastas dentro do AD para organizar os objetos. Assim, você pode aplicar uma regra de segurança só para o pessoal do Financeiro e outra diferente para as Vendas!
