A MURALHA DE FOGO

Juninho, qual é a principal função de um Firewall em uma rede empresarial?

📚 RESUMO PARA O JUNINHO:

O **Firewall** é como um segurança de balada. Ele analisa cada pacote de dados e decide se ele tem permissão para entrar ou sair da rede baseado em regras que você, o Analista, define!

CUIDADO COM A ISCA

Como se chama o ataque onde o hacker envia um e-mail falso para roubar senhas?

📚 RESUMO PARA O JUNINHO:

**Phishing** vem de "pescaria". O hacker joga uma isca (e-mail do banco, promoção falsa) para o usuário "morder" e digitar seus dados pessoais. É o ataque mais comum hoje!

O SEQUESTRO DE DADOS

Qual o nome do vírus que "tranca" os arquivos do cliente e pede um resgate em dinheiro?

📚 RESUMO PARA O JUNINHO:

O **Ransomware** criptografa os arquivos e exige pagamento (geralmente em Bitcoin). A melhor defesa para o seu cliente é ter um **Backup em Nuvem** sempre atualizado!

CHAVE MESTRA

Além de uma senha longa, qual tecnologia adiciona uma segunda camada de proteção?

📚 RESUMO PARA O JUNINHO:

O **MFA (Multi-Factor Authentication)** exige que o usuário confirme o login por um código no celular ou digital. Mesmo que o hacker descubra a senha, ele não consegue entrar sem o segundo fator!
