🔢 A REGRA DE OURO 3-2-1

Juninho, o que significa a regra 3-2-1 do Backup profissional?

📚 RESUMO PARA O JUNINHO:

A regra **3-2-1** é infalível: Tenha **3** cópias dos dados, em **2** tipos de armazenamento diferentes (ex: HD e Nuvem) e mantenha **1** cópia fora da empresa. Se a empresa pegar fogo, o dado está salvo!

💾 BACKUP INTELIGENTE

O que é um Backup Incremental que o Juninho configura no servidor?

📚 RESUMO PARA O JUNINHO:

O **Backup Incremental** economiza tempo e espaço. Em vez de copiar 1TB todo dia, ele só copia os arquivos novos ou alterados. É mais rápido e não sobrecarrega a rede da empresa!

🚨 O QUE É DR?

Qual o objetivo do Disaster Recovery (Recuperação de Desastres)?

📚 RESUMO PARA O JUNINHO:

**Disaster Recovery** é o plano de guerra. Se o servidor principal "morrer", você tem um segundo servidor (ou na nuvem) pronto para assumir o lugar. O objetivo é que o cliente nem perceba que houve um problema!

✅ O TESTE FINAL

Qual o erro mais comum que um Analista iniciante comete no Backup?

📚 RESUMO PARA O JUNINHO:

Backup que não foi testado não existe! O Juninho precisa, uma vez por mês, tentar **restaurar** um arquivo. Se o backup estiver corrompido e você só descobrir na hora da emergência, já é tarde demais!
