🧠 O CÉREBRO DO PC

Juninho, qual é a peça que processa todas as informações e comandos do sistema?

📚 RESUMO PARA O JUNINHO:

A **CPU (Unidade Central de Processamento)** é o motor! Ela faz os cálculos e dita a velocidade das tarefas. Sem um bom processador, a internet pode ser rápida, mas o PC vai travar!

⚡ MEMÓRIA DE CURTO PRAZO

Qual peça permite que o Juninho abra várias abas do navegador ao mesmo tempo sem travar?

📚 RESUMO PARA O JUNINHO:

A **RAM** é uma memória ultra-rápida que guarda o que está sendo usado *agora*. Quanto mais RAM, mais programas o Juninho consegue rodar juntos com alegria!

🗄️ O DEPÓSITO DE DADOS

Qual é a tecnologia de armazenamento mais moderna e rápida para ligar o Windows em segundos?

📚 RESUMO PARA O JUNINHO:

O **SSD** é muito superior ao HD antigo pois não tem peças móveis. Ele lê os dados via chips, sendo até 10x mais rápido que um HD comum. É essencial para um Analista hoje!

🔌 A MÃE DE TODAS AS PEÇAS

Onde o processador, a memória e o SSD são todos conectados?

📚 RESUMO PARA O JUNINHO:

A **Placa-Mãe** é o sistema nervoso do computador. Ela interliga todos os componentes e permite que eles conversem entre si. É a base de todo o hardware!
