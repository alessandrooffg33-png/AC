📦 MÁQUINAS VIRTUAIS

Juninho, o que é uma Máquina Virtual (VM)?

📚 RESUMO PARA O JUNINHO:

Uma **Máquina Virtual (VM)** é um computador "de mentirinha" que usa os recursos (RAM, CPU) do seu computador real. Com ela, o Juninho pode rodar Linux dentro do Windows com total segurança!

⚙️ O MAESTRO DAS VMS

Como se chama o software (como VMware ou VirtualBox) que gerencia as máquinas virtuais?

📚 RESUMO PARA O JUNINHO:

O **Hypervisor** é o software que "fatia" o seu hardware real e entrega para as VMs. Existem o Tipo 1 (direto no hardware, como o Proxmox) e o Tipo 2 (roda como programa, como o VirtualBox)!

🛡️ O PODER DO SNAPSHOT

Qual a maior vantagem de usar um "Snapshot" antes de testar um vírus ou comando perigoso na VM?

📚 RESUMO PARA O JUNINHO:

O **Snapshot** é como um "Save Point" de jogo. Se o Juninho fizer uma bobagem e o sistema travar, basta restaurar o Snapshot e a VM volta exatamente para o estado em que estava antes do erro. É pura magia!

🌐 MODO BRIDGE VS NAT

Qual modo de rede faz a VM aparecer como se fosse um PC físico real na rede da empresa?

📚 RESUMO PARA O JUNINHO:

No **Modo Bridge**, a VM usa a placa de rede do PC real para pegar o seu próprio IP no roteador. Ela vira um "vizinho" do seu computador real na rede local!
