🖥️ O GRANDE MAESTRO

Juninho, qual a principal função de um Sistema Operacional no computador?

📚 RESUMO PARA O JUNINHO:

O **Sistema Operacional (S.O.)** é o chefe! Ele serve como ponte entre você e as peças do PC. Sem ele, você teria que digitar códigos matemáticos complexos só para abrir uma pasta!

🐧 O MUNDO LINUX

Por que muitos Analistas de Redes preferem o Linux para gerenciar servidores?

📚 RESUMO PARA O JUNINHO:

O **Linux** é o rei da internet! Ele é robusto, raramente trava e a maioria das versões (distros) é gratuita e de código aberto. Quase todos os sites que você visita rodam em Linux!

🪟 WINDOWS NO TRABALHO

Qual ferramenta do Windows o Juninho usaria para ver as configurações de IP da rede?

📚 RESUMO PARA O JUNINHO:

O **CMD** é o seu melhor amigo no Windows! Usando comandos como `ipconfig`, você descobre tudo sobre a conexão do cliente em segundos. Analista de verdade domina a telinha preta!

📂 ONDE OS ARQUIVOS MORAM?

Como se chama a estrutura que o S.O. usa para organizar os arquivos no SSD?

📚 RESUMO PARA O JUNINHO:

O **Sistema de Arquivos** é a organização. O Windows usa muito o **NTFS**, enquanto o Linux prefere o **EXT4**. É como se fossem diferentes métodos de arquivamento em uma biblioteca!
