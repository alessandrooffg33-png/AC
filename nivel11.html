📶 AS DUAS ESTRADAS

Juninho, qual a diferença principal entre o Wi-Fi 2.4GHz e o 5GHz?

📚 RESUMO PARA O JUNINHO:

O **2.4GHz** é como um caminhão: lento, mas atravessa paredes fácil. O **5GHz** é como um carro de corrida: ultra rápido, mas qualquer parede o faz parar. Analista esperto usa 5GHz perto do roteador!

📻 O TRÂNSITO NOS CANAIS

Quais são os 3 únicos canais na frequência 2.4GHz que não se sobrepõem (não batem um no outro)?

📚 RESUMO PARA O JUNINHO:

No 2.4GHz, os canais são "largos". Se você usa o canal 2, ele bate no 1 e no 3. Para ter sinal limpo, os analistas só usam **1, 6 ou 11**. É o segredo para o Wi-Fi do vizinho não travar o seu!</
