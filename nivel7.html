>_ TESTE DE CONEXÃO

Juninho, qual comando você usa para saber se um site está "vivo" e respondendo?

📚 RESUMO PARA O JUNINHO:

O comando **ping** envia um pequeno sinal para um endereço e espera a volta. Se o sinal volta, a conexão está ok! É o primeiro teste que todo Analista faz.

>_ MEU ENDEREÇO

No Windows, qual comando mostra o IP da máquina e o Gateway da rede?

📚 RESUMO PARA O JUNINHO:

O **ipconfig** é o comando básico do Windows para ver as configurações de rede. No Linux, o comando equivalente (antigamente) era o **ifconfig**, mas hoje usamos mais o **ip a**.

>_ DIRETÓRIOS NO LINUX

Qual comando o Juninho usa no terminal Linux para "entrar" em uma pasta?

📚 RESUMO PARA O JUNINHO:

O comando **cd** significa *Change Directory* (Mudar de Diretório). Ele funciona tanto no Windows quanto no Linux para navegar entre as pastas!

>_ RASTREANDO O CAMINHO

Qual comando mostra por quais roteadores o sinal passa até chegar ao destino?

📚 RESUMO PARA O JUNINHO:

O **tracert** é incrível! Ele mostra cada "pulo" que seu dado dá na internet. Se a internet está lenta, ele te mostra exatamente em qual cidade ou roteador está o atraso!
