🔌 O CONECTOR PADRÃO

Juninho, qual o nome do conector padrão usado na ponta dos cabos de rede azul comum?

📚 RESUMO PARA O JUNINHO:

O conector **RJ45** é o rei das redes locais! Ele possui 8 pinos de metal que se conectam aos 8 fios internos do cabo de rede. É ele que você encaixa no PC e no Roteador.

🧶 CATEGORIAS DE CABO

Qual categoria de cabo é recomendada para redes que precisam de 1000 Mbps (Gigabit)?

📚 RESUMO PARA O JUNINHO:

Para internet de alta velocidade, usamos cabos **Cat 5e** ou **Cat 6**. O Cat 6 é ainda melhor porque tem uma proteção interna que evita interferências de outros cabos elétricos!

🎨 CORES DOS FIOS

Juninho, qual o padrão de cores mais usado no Brasil (T568B) para começar o cabo?

📚 RESUMO PARA O JUNINHO:

O padrão **T568B** é o mais comum: 1.Branco/Laranja, 2.Laranja, 3.Branco/Verde, 4.Azul, 5.Branco/Azul, 6.Verde, 7.Branco/Marrom, 8.Marrom. Seguir a ordem é o que garante que o sinal passe!

🔨 HORA DE APERTAR

Qual o nome da ferramenta usada para prender o conector RJ45 no cabo?

📚 RESUMO PARA O JUNINHO:

O **Alicate de Crimpagem** serve para "esmagar" os contatos de metal do RJ45 contra os fios de cobre. Se não apertar direito, a rede vai ficar caindo toda hora. Tem que ter força e jeito!
